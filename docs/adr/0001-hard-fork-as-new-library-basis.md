# Hard fork as the new library basis

This repository will be treated as a hard fork of upstream Qlementine, not as an upstream-compatible wrapper or patch set. The fork may rename the project, change namespaces and CMake targets, remove Qlementine-branded widgets, and give up long-term upstream synchronization because the planned Qt 5.15.2 port and widget/style redesign would make upstream compatibility costly and misleading.
